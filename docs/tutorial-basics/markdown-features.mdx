# 🤔 ¿Cómo inspeccionar y depurar?

Cuando mezclas Markdown con componentes React (como hace MDX en Docusaurus), el **Inspector de Elementos** de tu navegador se convierte en una herramienta fundamental para analizar cómo se renderizan estos componentes y cómo se comportan en tiempo real.

## Inspeccionando componentes MDX

Los componentes como `<Highlight />` que usas dentro del `.mdx` no son elementos HTML puros, sino **React components que generan HTML dinámicamente**.

Para ver cómo se visualizan:

1. Abre el sitio en el navegador.
2. Haz clic derecho sobre el componente y selecciona “Inspeccionar”.
3. Verás el HTML generado en el panel del DOM, con clases, estilos y eventos que han sido aplicados por React.

Esto es útil para:

- Ver estilos en línea aplicados dinámicamente.
- Ver qué `props` se convierten en atributos HTML.
- Confirmar si el componente se renderiza correctamente.

## Depura la interacción en tiempo real

En este ejemplo de componente:

```jsx
export const Highlight = ({ children, color }) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: "20px",
      color: "#fff",
      padding: "10px",
      cursor: "pointer",
    }}
    onClick={() => {
      alert(`You clicked the color ${color} with label ${children}`);
    }}
  >
    {children}
  </span>
);
```
